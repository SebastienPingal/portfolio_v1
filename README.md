 # Portfolio Version 1

 This project is my personal portfolio, showcasing my work as a fullstack developer and designer. It's built with Next.js, Prisma, and TailwindCSS for a modern, responsive design.

 ## Features

 - Dynamic project showcase
 - Blog functionality
 - Dark mode support
 - Responsive design

 ## Technologies Used

 - Next.js
 - Prisma
 - TailwindCSS
 - TypeScript
